# AI生成PPT - 智能PPT制作工具

一个现代化的 AI 驱动的 PPT 生成与编辑工具，三阶段式设计流程。

## 三阶段设计

### 阶段一：需求输入首页
- 标题"AI智能生成PPT"，红色主色调
- Word转PPT / 美化PPT / 图片转PPT 三种模式
- 热门场景标签、文件导入、尺寸/页数设置
- 4大智能功能（图片生成、视频生成、内容提炼、智能配图）
- 底部优势展示（高效智能、专业美观、多场景适用、安全可靠）

### 阶段二：PPT画布总览
- 左侧深灰色工具图标栏
- 顶部编辑工具栏（插入/文本/形状/图表/表格/媒体/主题/布局）
- 8张PPT幻灯片缩略图网格（各色渐变封面）
- 右侧属性面板（页面布局 / AI改稿 / 一键换图 / 样式调整）
- 底部AI提示输入栏 + 快捷操作按钮

### 阶段三：单页编辑模式
- 选中幻灯片放大展示
- 元素可点击选中（红色边框 + 调整手柄）
- 弹出AI搜索框（"告诉我你想要的需求"）
- 右侧动态属性面板

## 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite (Rolldown)
- **样式方案**: Tailwind CSS 3
- **图标库**: Lucide React
- **UI组件**: Radix UI
- **路由**: React Router v7
- **动画**: Motion (Framer Motion)
- **部署**: Vercel

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```
