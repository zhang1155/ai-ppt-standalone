# 需求文档

## 1. 应用概述

### 1.1 应用名称
AI生成PPT

### 1.2 应用描述
一个AI智能PPT生成网站,用户通过输入主题和需求,由AI快速生成专业PPT。采用现代简约的卡片式设计,左侧固定导航栏配合右侧主内容区的布局,提供一句话生成、Word转换、PPT美化等核心功能。

## 2. 用户与使用场景

### 2.1 目标用户
- 需要快速制作PPT的职场人士
- 对PPT设计要求较高但缺乏设计能力的用户
- 需要批量生成演示文稿的团队用户

### 2.2 核心使用场景
- 工作汇报、述职竞聘等职场场景的PPT制作
- 营销推广、产品发布等商务演示需求
- 教学培训、学术报告等教育场景

## 3. 页面结构与功能说明

### 3.1 页面结构
```
首页(单页应用)
├── 左侧导航栏(固定,宽200px)
│   ├── Logo区
│   ├── 导航菜单
│   ├── 功能按钮
│   └── 底部区域
└── 右侧主内容区
    ├── 顶部标题区
    ├── 核心输入区
    ├── 快捷功能区
    ├── 推荐模板区
    └── 右侧悬浮区
```

### 3.2 左侧导航栏

#### 3.2.1 Logo区
- 展示红色圆形图标
- 显示「创视PPT」文字
- 右上角显示红色标签「AI 智能生成」

#### 3.2.2 导航菜单
包含以下菜单项,每项配有对应图标:
  1. 首页(默认选中状态,红色高亮显示)
  2. AI生成PPT
  3. 模板中心
  4. 我的文档
  5. 历史记录
  6. 回收站
  7. 团队协作

#### 3.2.3 功能按钮
- 红色主按钮「开始生成」,采用圆角设计,醒目突出

#### 3.2.4 底部区域
- 显示消息、帮助中心、设置三个功能入口(图标+文字)
- 显示用户头像及用户名「晶晶」

### 3.3 右侧主内容区

#### 3.3.1 顶部标题区
- 大标题:「一句话 做PPT」
- 副标题:「输入主题和需求,AI 帮你快速生成专业PPT」
- 右侧装饰:魔法棒/星星图标,营造AI魔法感

#### 3.3.2 核心输入区
采用白色卡片样式,包含:
- 标签切换:一句话做PPT | Word转PPT | 美化PPT(三个tab)
- 大输入框,占位符文字:「请输入你想要PPT主题和内容,例如:人工智能发展现状与趋势...」
- 底部操作按钮:
  - 智能生成(红色主按钮)
  - 上传文件
  - 导入大纲
  - 选择模板
- 右下角红色麦克风按钮(语音输入功能)

#### 3.3.3 快捷功能区
6个图标网格布局,包含:
  1. AI大纲生成
  2. 智能排版
  3. 一键换肤
  4. 智能图表
  5. 模板推荐
  6. 语音输入

#### 3.3.4 推荐模板区
- 标题「推荐模板」,右侧红色「全部」标签
- 分类标签:工作汇报、述职竞聘、营销推广等
- 6张模板卡片网格展示(红色系商务风格)
- 每张卡片包含:缩略图、标题、描述

#### 3.3.5 右侧悬浮区
- 客服/帮助图标按钮
- 用户头像快捷入口

## 4. 设计规范

### 4.1 布局规范
- 左侧导航栏固定宽度200px
- 右侧主内容区自适应
- 窄屏时左侧导航可折叠为图标模式

### 4.2 视觉风格
- 主色调:#E53935(珊瑚红/红色)
- 背景色:白色
- 辅助色:浅灰色
- 圆角:8-12px
- 卡片轻微阴影效果
- 字体:中文无衬线字体(PingFang SC / Microsoft YaHei),标题加粗

### 4.3 交互规范
- 红色用于重点突出(按钮、标签、选中状态)
- 图标采用线性风格
- 留白充足,呼吸感强
- 所有内容在单页内完整展示

## 5. 验收标准

1. 左侧导航栏宽度为200px,包含Logo区、7个导航菜单项、开始生成按钮及底部功能区
2. 右侧主内容区包含顶部标题区、核心输入区、快捷功能区、推荐模板区及右侧悬浮区
3. 核心输入区包含3个tab切换、大输入框、4个底部按钮及右下角麦克风按钮
4. 快捷功能区展示6个功能图标网格
5. 推荐模板区展示6张模板卡片,每张卡片包含缩略图、标题、描述
6. 主色调为#E53935,圆角为8-12px,卡片带轻微阴影
7. 首页导航菜单默认选中并红色高亮
8. 窄屏时左侧导航可折叠为图标模式
9. 所有内容在单页内完整展示,无需滚动切换页面
